# 15 puzzle

A 15 puzzle built with react, typescript and tailwind.

# Installation

1. Clone repository

```bash
git clone https://github.com/sirisayshello/15-puzzle.git
```

2. Install dependencies

```bash
npm install
```

3. To set rows and columns go to `src/config.ts`

```ts
export const config = {
  rows: 4,
  columns: 4,
};
```

4. Start server

```bash
npm run dev
```

5. Have fun!
